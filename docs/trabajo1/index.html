Entrega examen Taller Diseño de interfaces

Descripción

En esta entrega para el examen de taller, creamos un mecanismo el cual según las diversas condiciones climáticas o flujos de aire, cerrara o abriera las ventanas de toda la casa. Está enfocado en los mayores de tercera edad, los cuales han perdido su movilidad articular y así poder facilitarles la vida diaria. Si no se quisiera tener el programa en modo automático, existían 2 botones en cada ventana los cuales uno de ellos desactivaba el sistema mientras que el otro abría y cerraba las ventanas.

Entrega Átomo al Bit

Descripción

En este trabajo se buscaba representar la gravedad por medio del programa Unity 3D, utilizando una pelota como protagonista. La contextualicé en un campo donde existían turbinas eólicas. Le agregué elementos fantásticos para que así la pelota pudiera tener posibilidades de movimiento, como resbalines y una especia de ascensor.
